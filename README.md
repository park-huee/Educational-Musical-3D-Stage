# Educational-Musical-3D-Stage
블렌더5.0을 활용하여 교육뮤지컬 퍼포먼스 및 효과를 3D 배경막에 담는 프로젝트를 진행한다.

## Tools Used
- Blender
- Eevee / Cycles
- 1920x1080 output for LED stage

## Stage Concept
- Dynamic camera movement
- Immersive lighting design
- Optimized for live performance screen

## Development Log
This repository documents the full production process including:
- Lighting setup experiments
- Camera path adjustments
- Rendering optimization
- Problem-solving records

## Goal
To build a performance-ready 3D background system applicable to real stage environments.
